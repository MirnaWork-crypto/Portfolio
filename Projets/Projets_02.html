01 — Description

À propos du projet

JavaStock est une application de gestion de stock développée en Java avec une interface graphique JavaSwing. L'application permet aux entreprises de gérer efficacement leurs inventaires, leurs commandes et leurs relations avec les fournisseurs.

Le projet met en œuvre les principes de la programmation orientée objet (POO) avec une architecture MVC claire, une connexion à une base de données PostgreSQL via JDBC, et une interface utilisateur intuitive développée avec JavaSwing.

02 — Contexte

Objectifs du projet

Ce projet avait pour objectifs principaux :

  • Développer une application desktop robuste en Java
  • Mettre en pratique les concepts de POO (encapsulation, héritage, polymorphisme)
  • Concevoir et implémenter une architecture MVC (Model-View-Controller)
  • Manipuler une base de données relationnelle PostgreSQL
  • Créer une interface graphique ergonomique avec JavaSwing
  • Gérer les erreurs et valider les données utilisateur
